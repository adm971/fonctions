# fonctions
Petit programme développé en python pour représenter des fonctions affines et quadratiques


Installation 
Sur Linux
1. Ouvrir l'invite de commande 

2. sudo apt update && sudo apt upgrade && sudo apt install python3 && sudo apt install python3-pip && pip3 install cx-freeze && git clone https://github.com/adm971/fonctions.git && cd fonctions && ls && python3 setup.py 

3. un dossier "build" sera généré, l'ouvrir et chercher le programme "main" à lancer


Une fois le programme lancé 
Il est possible de représenter des fonctions affines ou quadratiques soit en entrant soit meme les valeurs de x, a, b, ou c soit en 
laissant les cases vides pour laisser le programme générer aléatoirement les fonctions
